# 基於 LLM 的雙向手語虛擬人翻譯系統

這是一個可直接用 VS Code 開啟的前端展示版本，包含兩大畫面：

1. 雙向虛擬人物辨識與辨識文字
2. 文字生成虛擬人物手語

## 使用方式

1. 用 VS Code 開啟此資料夾
2. 直接用瀏覽器開啟 `index.html`
3. 或使用 VS Code 的 Live Server 執行

## 功能說明

- 左側畫面：
  - 模擬虛擬人物手語辨識
  - 顯示辨識進度
  - 輸出辨識後的中文文字

- 右側畫面：
  - 輸入中文句子
  - 使用 LLM 翻譯概念生成手語動作流程
  - 用 Canvas 模擬虛擬人物手語播放

## 後續可擴充

- 串接 webcam 即時影像
- 串接 MediaPipe 骨架辨識
- 串接 OpenAI API 或本地 LLM
- 改接 3D 虛擬人物模型
- 匯入真實台灣手語動作資料集

## 注意

目前版本為專題展示 prototype，重點是畫面完整、流程清楚、可展示雙向翻譯概念。
